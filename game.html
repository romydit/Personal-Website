Game in Processing
I created a mini-game using Processing, I learned how to write methods like keyPressed() and mouseReleased() to ensure that players could interact with the game. I drew the components using chalk pastel, oil pastel, and colored pencil and then touched up the images in Photoshop. To export the project I converted the Processing code into p5.js.
