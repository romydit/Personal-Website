childrens' piano (2026)
Media: p5.js, Photoshop, and Audify
At the end of my Digital Media course, I developed a childrens' animal keyboard using the coding program Processing. I combined the concepts I had learned in Processing like interactive methods such as MouseClicked() and MousePressed(). I made the project multimodal by recording the animal sound effects using the Software Audacity and used the Minim and AudioOutput libraries for the piano keys. In order make the project accessible on the web I changed the syntax into p5.js code using polySynth for the piano keys. The design was created in Photoshop.
