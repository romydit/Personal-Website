Childrens' piano (2026)
Mini-game in Processing (2026)
childrens' piano Digital Media coursework, 2026
At the end of my Digital Media course, I developed
a childrens' animal keyboard using the creative coding
program Processing. I combined the concepts I had learned
in Processing like interative methods such as MouseClicked()
and MousePressed() and creating Class instances with
the Audio Software Audacity to record audio to make my
project multimodal. The green piano keys can play an octave
from C4 to C5 using the Minim and AudioOutput libraries.
I also recorded four animal sounds using the software.
Audacity and imported them into my project using the
AudioSample library. In order make the project accessible
on the web I changed the syntax into p5.js code.
